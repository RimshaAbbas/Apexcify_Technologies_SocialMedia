# Apexcify_Technologies_SocialMedia
Mini Social Media Platform is a lightweight, fully functional social media web app built as a learning/project showcase. It allows users to create profiles, post content, like posts, comment, and follow other users — all with a clean, responsive interface using vanilla HTML, CSS, and JavaScript on the frontend and Express.js + SQLite on the backend.
Core Features:

User registration/login (session-based auth)
Create, edit, delete posts
Like/unlike system with live counts
Comments on posts
Follow/unfollow users
User profiles with bio, avatar, and activity

No heavy frameworks — pure and simple.
